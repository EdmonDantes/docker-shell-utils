# Docker Shell Utilities

Repository with shell utilities for Docker images.

## Variants

### Development

This variant contains utils for development docker images. It requires `docker` for correct work.

You can find the full description [here](dev/README.md)

### Debian-based

This variant contains utilities for Debian-based Docker images (based on Debian or Ubuntu). It requires `bash` for
correct work.

You can find the full description [here](debian-based/README.md)

### Alpine

This variant contains utils for Alpine Docker images (based on Alpine)

You can find the full description [here](alpine/README.md)
